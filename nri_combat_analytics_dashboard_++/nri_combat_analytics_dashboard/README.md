# nri_combat_analytics_dashboard

## Описание

Репозиторий содержит веб приложение для визуализации и анализа данных по игровым сессиям, персонажам, боевым столкновениям и способностям в системе "НРИ-Помощник (офлайн)".

Приложение предоставляет единый интерфейс для просмотра журнала событий, сводных показателей, таблиц и графиков. Основное внимание уделяется персонажам, урону, попаданиям, критическим успехам, структуре способностей и характеристикам игровых сессий.

## Состав данных

### characters.csv
Содержит сведения о персонажах и NPC: имя, роль, класс защиты, запас здоровья, модификатор инициативы и принадлежность к категории Player или NPC.

### encounters.csv
Содержит список боевых столкновений, связанных с игровыми сессиями и сценами.

### combat_events.csv
Содержит журнал боевых событий: атакующий, цель, способность, тип урона, величина урона, итог броска, класс защиты цели, факт попадания и признак критического успеха.

### abilities.csv
Содержит список способностей персонажей с указанием типа ресурса, базовой силы и числа использований за сессию.

### sessions.csv
Содержит сведения об игровых сессиях: код сессии, кампания, дата проведения, длительность, число столкновений и количество заметок.

## Разделы интерфейса

### Дашборд
Содержит фильтры и сводные показатели по боевым событиям, а также графики суммарного урона, динамики по раундам, структуры урона и частоты попаданий.

### Персонажи
Содержит карточки показателей по составу персонажей, диаграмму соотношения класса защиты и запаса здоровья, а также таблицу со справочными данными.

### Боевые события
Содержит фильтр по столкновению и номеру раунда, график хронологии действий и таблицу журнала событий.

### Способности
Содержит графики по распределению способностей, их базовой силе и типам ресурсов, а также таблицу со списком способностей.

### Сессии
Содержит графики по длительности сессий, количеству заметок и числу столкновений, а также таблицу по проведенным игровым сессиям.

### Аналитика
Содержит режимы просмотра данных по точности атак, интенсивности урона и критическим успехам.
